Notepad.exe template por Gian Peres

Olá, obrigado por visitar meu site. Eu acabei de criá-lo e logo ele terá uma cara definitiva.

Este modelo foi feito exclusivamente para se parecer com o bloco de notas, por que você gostaria de uma página da web como esta ... bem, para a maioria das empresas, eu diria que provavelmente não funcionaria. Mas para aqueles de vocês que são um pouco peculiares e querem uma página da web que definitivamente receberá uma aparência de alguns segundos, é isso que você está procurando.

É totalmente XHTML 1.1 válido (o mais novo, mais rigoroso até o momento), a navegação é totalmente funcional e idêntica ao bloco de notas em seus rollovers e fontes. Todas as fontes são iguais. Além disso, projetei este modelo para que você possa facilmente alterar o logotipo no canto superior direito para outra coisa. Tudo que você precisa fazer é criar uma imagem 16x16 e alterar o arquivo CSS (as instruções estão em style.css)

Sinta-se à vontade para mudar o plano de fundo também, isso provavelmente funcionaria muito bem com um plano de fundo de alta resolução, como papel de parede. E provavelmente seria ainda mais legal se o plano de fundo tivesse alguns ícones da área de trabalho intactos.

Tags H1 são os únicos elementos em negrito

Eu realmente não queria usar nenhum elemento em negrito, mas percebi que, como a maioria das pessoas deseja usar títulos de página, eu faria as versões em negrito de h1 do texto normal. Fiz isso por esse motivo, e vai te ajudar também com os motores de busca (SEO).
